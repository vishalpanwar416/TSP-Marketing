# Social Media & Marketing App

Standalone application for social media automation and marketing (campaigns, contacts, WhatsApp, email, templates, scheduling). Built from the Marketing Dashboard.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Same Firebase

Uses the same Firebase project as the parent certificate app (`config/firebase.js`). Update that file if you need a different project.
